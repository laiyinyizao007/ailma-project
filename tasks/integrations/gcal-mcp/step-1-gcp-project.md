# Step 1: 创建 GCP 项目

**耗时**: 5 分钟 | **状态**: ✅ 已完成

---

## 🎯 目标

在 Google Cloud Platform 创建项目，用于访问 Calendar API。

---

## 📋 子步骤

### 1.1 访问 GCP Console (1 分钟)

- [ ] 打开 https://console.cloud.google.com
- [ ] 登录 Google 账号

**检查点**: 看到 GCP 控制台

---

### 1.2 创建新项目 (2 分钟)

- [ ] 点击顶部项目选择器
- [ ] 点击 "New Project"
- [ ] 填写信息:
  - Project name: `AILMA`
  - Location: (默认)
- [ ] 点击 "Create"

**检查点**: 项目创建成功

---

### 1.3 记录项目 ID (1 分钟)

- [ ] 选择刚创建的项目
- [ ] 在项目信息卡片中找到 "Project ID"
- [ ] 复制并保存

```
Project ID: ailma-xxxxx
```

**检查点**: 项目 ID 已记录

---

### 1.4 设置默认项目 (1 分钟)

- [ ] 确保顶部显示 "AILMA" 项目
- [ ] 如果不是，点击选择器切换

**检查点**: 当前项目为 AILMA

---

## ✅ 完成标准

- [x] GCP 项目已创建
- [x] 项目 ID 已记录
- [x] 项目已选中
