# AILMA - AI 智能生活管理助手

**一句话介绍**: 用自然语言管理你的日历和笔记，AI 自动生成工作报告。

---

## 🎯 产品定位

AILMA (AI Life Management Assistant) 是一个**零界面**的智能助手：

- 📝 **Notion 作为前端** - 在熟悉的 Notion 中发出指令
- 🤖 **AI 理解意图** - 用口语化的文字描述任务
- ⚡ **自动执行** - 后台自动操作日历、创建笔记、生成报告
- 🔗 **MCP 驱动** - 使用 Model Context Protocol 统一集成

---

## 💡 解决什么问题？

### 问题 1: 平台分散 ❌

```
日历在 Google Calendar
笔记在 Notion
任务在 Todoist
报告手动整理
```

**→ 需要频繁切换平台，信息孤立**

### 问题 2: 操作繁琐 ❌

创建一个会议需要：
1. 打开 Google Calendar
2. 点击时间槽
3. 填写标题、时间、参会者
4. 保存
5. 打开 Notion
6. 创建会议纪要页面
7. 手动关联日历链接

**→ 7 个步骤，5 分钟**

### 问题 3: 报告耗时 ❌

生成周报需要：
- 查看日历统计本周会议
- 查看 Notion 统计完成任务
- 手动汇总数据
- 写 Markdown 格式报告
- 发布到 Notion

**→ 30 分钟手动劳动**

## ✅ AILMA 的解决方案

### 解决方案 1: 统一入口 ✅

```
在 Notion "指令中心"输入：
"明天下午3点和团队开会讨论Q1规划"
```

**→ AILMA 自动**：
- ✅ 创建 Google Calendar 事件
- ✅ 生成 Notion 会议纪要页面
- ✅ 关联日历链接
- ✅ 生成 Google Meet 链接

**1 条指令，10 秒完成**

### 解决方案 2: 自然语言 ✅

```
普通话输入，AI 理解：

"明天下午3点"         → 2025-11-28T15:00:00+08:00
"下周五"              → 2025-12-06
"每周一上午10点"      → 重复事件
```

**→ 无需记忆复杂命令格式**

### 解决方案 3: 自动生成报告 ✅

```
在 Notion 输入：
"生成本周工作报告"
```

**→ AILMA 自动**：
- ✅ 从 Google Calendar 获取本周会议
- ✅ 从 Notion 获取完成任务
- ✅ AI 分析和总结
- ✅ 生成 Markdown 格式周报
- ✅ 发布到 Notion "报告归档"

**1 分钟自动完成**

## 🌟 核心特性

### 1. Notion 作为界面

- ✅ 无需开发前端 UI
- ✅ 利用 Notion 强大的数据库和视图
- ✅ 用户在熟悉的环境中操作

### 2. MCP 驱动集成

- ✅ **Notion MCP**: Markdown 原生支持，98% 代码减少
- ✅ **Google Calendar MCP**: 自然语言时间解析，60% 代码减少
- ✅ **统一 OAuth**: MCP Server 托管认证，零维护

### 3. AI 驱动智能

- ✅ 自然语言意图识别（LangChain + Claude）
- ✅ 实体提取（时间、人名、地点）
- ✅ 智能报告生成

### 4. 自动化工作流

- ✅ 后台监听 Notion 指令
- ✅ 自动执行任务
- ✅ 结果回写到 Notion

## 🎬 使用示例

### 示例 1: 创建会议

**输入**（Notion 指令中心）:
```
明天下午3点和产品团队开会讨论新功能
```

**输出**:
- ✅ Google Calendar: 创建事件 "产品团队会议 - 新功能讨论"
- ✅ Notion: 创建会议纪要页面，包含议题模板
- ✅ 自动生成 Meet 链接

**状态更新**（Notion）:
```
[✅ Done] 已创建会议并生成纪要
相关链接: https://calendar.google.com/event/...
会议纪要: https://notion.so/page/...
```

### 示例 2: 查询日程

**输入**:
```
我下周有什么安排？
```

**输出**（Notion）:
```markdown
# 📅 下周日程（2025-12-02 ~ 2025-12-08）

## 周一 (12/02)
- 10:00-11:00 | 团队站会
- 14:00-15:30 | 产品评审

## 周三 (12/04)
- 15:00-16:00 | 客户演示

共 3 个事件，总时长 3.5 小时
```

### 示例 3: 生成报告

**输入**:
```
生成本周工作报告
```

**输出**（自动创建 Notion 页面）:
```markdown
# 📊 2025-W48 工作总结

## 📈 关键指标
- 会议时长: 12.5 小时
- 深度工作: 18 小时
- 完成任务: 15 项

## ✅ 本周完成
- [x] 完成产品需求文档
- [x] 实现用户认证模块
- [x] 3 次客户会议

## 📅 重要会议
- 2025-11-25 | 产品规划会议
- 2025-11-27 | 技术评审

## 💡 下周计划
- [ ] 上线用户认证功能
- [ ] 准备产品演示
```
